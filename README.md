# Scalable Analytics for Real-Time Traffic Detection, Prediction and Control

## Overview
This project presents a scalable real-time traffic analytics framework for smart city environments. It continuously processes traffic streams, predicts accident probability and severity using machine learning, computes congestion analytics, and performs intelligent traffic control with ambulance priority management.

## Features
- Real-time traffic data simulation
- Apache Kafka streaming (Producer–Consumer)
- Congestion index calculation
- Traffic phase detection
- Accident probability prediction
- Accident severity classification
- Adaptive traffic signal control
- Ambulance green corridor activation
- Streamlit dashboard for visualization
- SQLite database for processed traffic records

## Technology Stack
- Python
- Apache Kafka
- Streamlit
- SQLite
